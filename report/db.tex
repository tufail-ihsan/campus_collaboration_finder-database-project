\begin{abstract}
The Campus Collaboration Finder is a web-based platform that connects university students for collaborative projects through skill-based matching. Students can create profiles, post projects with required skills, apply to projects, and receive AI-powered recommendations. The system is built using Flask (Python) backend, MySQL database, and a responsive frontend. It provides full CRUD functionality along with a smart matching algorithm that calculates skill match percentage.
\end{abstract}

\begin{IEEEkeywords}
Skill-based Matching, Project Collaboration, Recommendation System, Flask API, MySQL, Web Application
\end{IEEEkeywords}

\section{Introduction}
Finding suitable team members for academic projects is often challenging for university students. This project aims to solve this issue by developing a centralized platform that matches students with projects based on their skills.
